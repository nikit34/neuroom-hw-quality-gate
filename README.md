# Fake Screen HW Quality Gate (prototype)

## Что внутри
- `index.html` — прототип экрана.
- `styles.css` — стили мобильного экрана.
- `script.js` — сценарий:
  1. Показывает `Проверяем качество...` на случайные 1-2 сек.
  2. Показывает 2 CTA: `Отправить` (primary) и `Переснять` (secondary).

## Как открыть
Откройте файл `index.html` в браузере.
